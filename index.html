Welcome to EventFinder. To navigate around, the best choise would be to login. There are three types of login users, admin, customer and host. As a customer you will be able to use the top left button "Booked Events", to check your previously booked events. To book an event immediately, please use the top right input search bar which will redirect your to different event choises.Hosts will be able to use the create event button on the top left only when logged in. New events can easily be created with the functionality of the website.
